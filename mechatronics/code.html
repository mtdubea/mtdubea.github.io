Code Overview

Detailed Description of Arduino Programs for Payload System

1. master_payload.ino

This is the main controller code. It integrates the three mission tasks:

  • Detecting nearby objects using ultrasonic sensors
  • Recognizing water via sensor and triggering a unique audio signal
  • Releasing a payload upon detecting elevated temperature

2. TiltAngle.ino

This program reads data from the MPU9250 accelerometer/gyroscope and calculates the tilt angle. It is used to track drone orientation or pitch/roll movements. 

// Sample snippet
float getTiltAngle() {
  float angle = atan2(accY, accZ) * 180 / PI;
  return angle;
}

3. Orientation.ino

This script calculates the real-time 3D orientation of the drone using sensor fusion from the MPU9250 gyroscope, accelerometer, and magnetometer. It may use complementary or Kalman filters. 

// Sample snippet
void computeOrientation() {
  roll = atan2(accY, accZ) * RAD_TO_DEG;
  pitch = atan2(-accX, sqrt(accY * accY + accZ * accZ)) * RAD_TO_DEG;
}
