Compilation & Flash Guide

  1. Install Arduino IDE 2.x; board = “Arduino Uno”.
  2. Clone repo or download ZIP → extract, leaving folder names unchanged.
  3. Open master_payload.ino. IDE auto‑loads library from /code.
  4. Tools › Port → select your COM/tty device, then Upload.

The code uses Serial.begin(115200) for debug prints – open Serial Monitor after flash to view live diagnostics.

Power Architecture

Logic rail – 6 × AA (≈9 V) regulated to 5 V for MCU / sensors. Actuator rail – dedicated 9 V pack isolates inductive kick from servo + buzzer. Spare 9 V secured on bracket for hot‑swap during field trials.
