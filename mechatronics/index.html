Mechatronics Final Project

University of Louisiana at Lafayette — Spring 2024

Overview

This capstone mechatronics project integrates mechanical, electrical, and software systems using sensors, actuators, and embedded control logic on an Arduino microcontroller. The system detects environmental stimuli and reacts autonomously in real time.

Key Components

  • Arduino Uno – core microcontroller for logic and control
  • MPU-9250 – IMU (Inertial Measurement Unit) for orientation & tilt display
  • Ultrasonic Sensor – obstacle detection with distance alert
  • Water Sensor – environmental hazard detection (moisture)
  • Thermistor – triggers payload release at preset temp
  • Servo Motor – controls payload door release
  • LCD Display – real-time sensor readouts (angle, direction)

Features & Logic

The system operates based on modular sensor logic:

  • Displays tilt/orientation via MPU-9250 on LCD
  • Triggers buzzer on water detection
  • Servo motor responds when thermal threshold is passed
  • Ultrasonic sensor alerts to nearby obstacles
