🔧 Summary of Core Functions

  • master_payload.ino: Executes all mission logic — object detection, water sensor alert, temp-triggered payload drop.
  • TiltAngle.ino: Calculates device tilt using MPU9250 accelerometer data (for balance/orientation).
  • Orientation.ino: Computes 3D orientation using sensor fusion (acc + gyro + mag).
  • task3-6.ino: Handles Tasks 3–6: distance alerts, water detection tone, heat-triggered deployment.
