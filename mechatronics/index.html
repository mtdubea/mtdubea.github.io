Drone-Based Mechatronics Payload

MCHE 357 · Spring 2024 · University of Louisiana at Lafayette

Project Overview

This project involved designing and building a drone-deployable Arduino payload system equipped with multiple sensors for real-time environmental and positional feedback. The system demonstrated autonomous sensor-triggered actuation and was field-tested during a live capstone demo.

Key Features

  • MPU-9250: Orientation + tilt displayed live to LCD
  • Ultrasonic Sensor: Object proximity alerts
  • Water Sensor: Audio beep code upon contact
  • Thermistor: Temperature-triggered servo release
  • Servo Motor: Mechanical control of payload system

Responsibilities

  • Designed, soldered, and tested sensor-actuator system on breadboard
  • Programmed Arduino code modules for each subsystem
  • Validated system logic under test scenarios
  • Presented working prototype with full documentation
